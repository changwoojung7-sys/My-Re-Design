# 📢 [업데이트] 서비스 개편 및 주요 기능 강화 안내

안녕하세요, **My Re Design** 사용자 여러분!
여러분의 성장을 더 효과적으로 지원하고 편의성을 높이기 위해 서비스가 대폭 업데이트되었습니다.
오늘 적용된 주요 변경 사항을 확인해 보세요.

### 1. 계정 및 보안 강화 🔐
*   **이메일 전용 회원가입**: 서비스 안정성을 위해 회원가입 방식을 **이메일**로 통합했습니다.
*   **보안 인증 도입**: **회원가입 및 탈퇴** 시 이메일 인증 절차를 추가하여 여러분의 계정을 더욱 안전하게 보호합니다.

### 2. 미션 시스템 전면 개편 🎯
*   **6대 핵심 영역 재구성**: 균형 잡힌 성장을 위해 미션 카테고리를 **6가지(Health, Growth, Mindset, Career, Social, Vitality)**로 체계화했습니다. 더욱 정교해진 맞춤형 성장을 경험해 보세요.

### 3. 소셜 및 편의성 강화 🤝
*   **친구 검색 개선**: 친구를 더 쉽게 찾을 수 있도록 검색 기능을 강화했습니다. 이제 **휴대폰 번호**나 **이메일 주소(전체)** 중 편한 방법으로 검색하면 바로 찾을 수 있습니다.
*   **구독 프로세스 개선**: 프리미엄 구독 결제 및 관리 절차를 더욱 명확하고 편리하게 재정립했습니다.

### 4. UI 디자인 최적화 ✨
*   **미션 화면 개선**: '진행 중' 상태와 '미션 개요'를 더 시원하고 명확하게 볼 수 있도록 디자인(폰트/아이콘 확대, 레이아웃 등)을 개선했습니다.
*   **디자인 통일**: 앱 내 모든 메뉴의 타이틀 스타일을 **'아이콘 + 영문'**으로 통일하여 일관성 있는 경험을 제공합니다.


### 5. 구독 및 결제 관리 기능 고도화 💳
*   **결제 취소 기능 추가**:
    *   **사용자**: 결제 당일에 한해 구독 관리 페이지에서 즉시 결제를 취소할 수 있습니다.
    *   **관리자**: 관리자 대시보드에서 기간 제한 없이 사용자의 결제를 취소할 수 있습니다.
*   **데이터 동기화 강화**: 결제 취소 시 해당 기간의 **구독 권한도 즉시 '취소됨(Cancelled)' 상태로 동기화**되도록 로직을 개선하여, 결제 상태와 프리미엄 권한의 불일치를 방지했습니다.
*   **직관적인 상태 표시**: 나의 구독 관리 및 관리자 페이지에서 취소된 결제 내역과 날짜를 명확하게 확인할 수 있습니다.

**My Re Design**은 여러분의 피드백을 바탕으로 매일 성장하고 있습니다.
새로워진 **My Re Design**과 함께 오늘도 목표를 달성해 보세요! 감사합니다.
